This is a simple 2048 game implemented in Python using the kivy library. Use the arrow keys to move the tiles and combine them to reach the 2048 tile. For undo use the 'ctrl+z' key. 

To run the game, make sure you have Python and the kivy library installed. You can install kivy using pip:

```pip install -r requirements.txt```

Then, run the main.py file to start the game:

```python main.py```
